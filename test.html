Problème de réception d'emails FormSubmit

Solutions aux problèmes courants de configuration

Vous ne recevez pas les emails de FormSubmit?

Plusieurs raisons peuvent expliquer pourquoi vous ne recevez pas les emails de confirmation ou les soumissions de formulaire.

Solutions possibles :

1 Vérifiez les spams

Les emails de FormSubmit peuvent parfois être classés comme spam. Vérifiez votre dossier de courriers indésirables.

2 Configuration DNS

Ajoutez un enregistrement TXT à votre domaine pour vérifier la propriété et améliorer la délivrabilité.

v=formsubmit-prove-ownership xxxxxxxx

3 Email de secours

Ajoutez un email de secours dans les paramètres de FormSubmit au cas où l'email principal ne fonctionnerait pas.

Vérification de votre configuration actuelle

Votre formulaire est configuré avec l'action :

action="https://formsubmit.co/school.dlecointre@gmail.com"

Cette configuration semble correcte. Le problème pourrait venir des serveurs de FormSubmit ou de votre fournisseur email.

Solutions alternatives

Si FormSubmit continue de poser problème, vous pouvez envisager ces alternatives :

  • Utiliser un service similaire comme GetForm ou Formspree
  • Configurer un script PHP personnalisé pour traiter les formulaires
  • Utiliser un service de backend comme Firebase ou AWS Lambda
